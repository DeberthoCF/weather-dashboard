# Weather Dashboard

## Description

This project is a responsive weather dashboard developed for the LOG3500 Web Programming course.

The application allows users to search for a city and displays current weather information retrieved from the Open-Meteo API.

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- Async / Await
- Open-Meteo API

## Features

- Search weather by city
- Responsive design
- Loading indicator
- Error handling
- Accessible form
- Semantic HTML
